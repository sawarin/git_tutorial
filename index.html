Gitチュートリアル

git clone <リポジトリ名>

gitリモートリポジトリ(github)のコピーをローカルリポジトリに作成する
→ git clone https://github.com/sawarin/git_tutorial.git

git add <ファイル名・ディレクトリ名・.>

ワークツリーの変更をステージに追加する
→ git add index.html

git commit

ステージの変更をローカルリポジトリに記録する
→git commit

git status

現在の状況を確認する
→git status

git diff

ワークツリーとステージとの差分(git addする前の変更分)
→git diff

git diff --staged

ステージとリモートリポジトリの差分(git addした後の変更分)
→git diff --staged

git log

変更履歴を確認する
→git log --online
→git log -p index.html
→git log -n 1
